$sock

$sock : integer

Socket file descriptor

Example

$sock = fopen("php://fd/".$swoole_client->sock);

The swoole_client is a stream socket. You can call fread / fwrite / fclose functions. For swoole_server, the $fd cannot use this method to convert as $fd is just a number, $fd file descriptor is part of the main process $swoole_client->sock can be converted to int as the key array.

connect()

connect(string  $host, integer  $port, float  $timeout = 0.1, integer  $flag) : boolean

Connect to the remote server

  • Before send / recv swoole_client_select must be used to detect if the connection is open

Parameters

string $host

is the address of the remote server, v1.6.10+ Swoole will automatically fill the domain DNS query
integer $port

is the remote server port
float $timeout

is network IO timeout, the unit is s, support for floating point. The default is 0.1s, ie 100ms
integer $flag

parameter when UDP type indicates whether udp_connect enabled. Setting this option will bind $host and $port, the UDP will discard non-designated host / port packet.

Returns

boolean

recv()

recv(integer  $size = 65535, boolean  $waitall = false) : string|boolean

Receive data from the server

If you set the $waitall you must set the correct $size, otherwise they will have to wait until the data length reaches $size is received      If you set the wrong $size, can cause recv timeout & returns false      Successful call returns the resulting string, failed to return false, and sets $swoole_client->errCode property

Parameters

integer $size

the maximum length of the received data
boolean $waitall

whether to wait for all the data $size

Returns

string|boolean
